タイトル「表裏一体」
ジャンル「横スクロールアクション」
使用エンジン「Unity」
開発期間「1ヵ月」

プロジェクトメンバー
・プランナー　4人
・プログラマー　2人
・グラフィック　1人

・概要
　このゲームは今年度のゲーム大賞に向けて制作途中のゲームとなって
おります。背景やUIなどが制作途中ですので、仮のものを使用しております。
ご了承ください。


・操作説明
　画面をタップすること（又はスマホを振る）で、上にあるオブジェクトは下へ、
下にあるオブジェクトは上へ移動します。
赤いオブジェクトは下側のキャラが殴って倒してくれます。
白いオブジェクトは上の少女がジャンプすることでよけられます。
